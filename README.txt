El juego Solitario tiene muchas variantes y este proyecto no pretende abarcar todas las posibilidades.
Este proyecto dispone de:
- 3 niveles de dificultad.
- Animaciones en la victoria o derrota.
- Ajustes de apariencia.
- Cambio de idioma entre español e inglés.

Y es un juego realizado en Front-End. (HTML, CSS, JavaScript / JQuery y un JSON para el cambio de idioma).
